Im Maximillion Chua and I wrote this chip-8 emulator in rust.
This project was made for fun.
Instrucions for running it,

1. First open up a code editor and in main.rs find -  let rom = rom::load("Tetris [Fran Dachille, 1991].ch8");
and edit the string to be whatever ch8 rom you want.
2.as i mentioned its written in rust, so you have to have rust downloaded on your computer.   https://www.rust-lang.org/tools/install
3.in your terminal navigate to the directory and enter cargo run 
4. enjoy!

This was an extremely fun and interesting project. I used minifb for my graphics+input api.
